Development of Fixed Asset Management

About the project

The customer wanted to develop a customized Fixed Assets Management System based on their specific provided requirements. This system is being developed on a productized model to be integrated with multiple products wherever possible. Mentioned below are some of the key features being developed:

Our solution

As part of the project the following modules were developed:

  • Create/Define the Asset Types
  • Create/Define locations
  • Create/Define Manufacturers
  • Create/Edit/Delete New Asset along with all detail, Attach the related document.
  • Asset Register along with comprehensive search where all the assets will be listed.
  • Activity Detail: Schedule activity for external or internal calibration.
  • Send notification to owner before few days of scheduled date.
Technologies/Tools
  • Angular 12 with Material UI
  • .Net Core
  • AG Grid
  • MS SQL Server
