Ezy Articles

About the Project

In EZY Articles Website stripe payment gateway is integrated for customers. So that they can make Top Up payments and can access the published papers through mobile apps. Third party users use this website to publish papers which are assigned to them. Others who are not recognized customers of the application can make other payments and then they will be registered as customers by admin from the admin site.

Stack Used

.net core MVC, Entity framework for Database integration, identity framework for user management, MySQL Database

Third Party Integrations

Stripe payment gateway integration, AWS Email Service Integrated

EZY Articles Admin Site

Admin and CSR are the users of this application, Admin can perform CRUD operations on customers, payment Options, Third party users, and can upload and validate papers, add new admin and CSR and can perform CRUD operations on them, can grant them permissions role wise as well as user wise. Admin can also assign papers of particular name, type and size to third party users. CSR can also perform these operations based on the access provided by the admin.
