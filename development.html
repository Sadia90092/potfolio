Number Plate Scanner

About the project

We have an existing application that has a c++ number plate recognition engine that captures video from an RTSP IP camera feed. This is wrapped in a C# application which provides additional features and functionality. We are looking for an experienced developer to make changes to the existing application add new functionality and improvements. You must be experienced in:

  • C++
  • C#
  • API's
  • Multithreading
  • Setup file creation
  • RTSP streaming

Plus all the other things associated:

We will dedicate a team member to help you understand the project, but we need someone who can start straight away and just get on with the tasks we need. Most importantly you need to be able to communicate and ask questions if you don't understand and do not apply if you are not experienced in this technology stack.
