About the Project

We need a Xamarin App based on C# language for both Android and iOS. The app will use our API methods and needs the using of a bearer token for each Https call. The API documentation is very detailed

  • User creation
  • User login
  • Manage the user data in order to permit the update
  • Renew the password
  • Qr Code viewer
About that feature we need
  • Simple list of the training cards
  • Detail of a training card with all the exercises linked
  • The possibility to see the trainer's notes in a pop-up and, if it is present, a video we have to show a video player where shown a YouTube video
  • The possibility for the user to add notes linked to an exercise and read it when he needs it
  • A training card doesn't have only exercises but also Circuit and Super Series which are simple rows in green and yellow as you can see in the attached images
